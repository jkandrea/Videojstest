# 블록툴툴 (BlogToolTool)

블로거를 위한 무료 온라인 이미지 편집 도구

🌐 **웹사이트:** [https://blogtooltool.com](https://blogtooltool.com)

## 🎯 주요 기능

### 🎬 GIF 변환기
- 동영상 파일(MP4, MOV, AVI 등)을 GIF로 변환
- FPS, 크기, 구간 조절 가능
- 실시간 파일 크기 확인

### 🔖 워터마크 추가
- 이미지에 워터마크 추가
- 위치, 투명도, 크기 조절
- 저작권 보호

### 🖼️ 블로그 썸네일
- 블로그 포스팅용 썸네일 제작
- 다양한 템플릿 제공
- 최적 크기 지원

## ✨ 특징

- ✅ **완전 무료** - 모든 기능 무료 제공
- ✅ **회원가입 불필요** - 바로 사용 가능
- ✅ **개인정보 보호** - 파일이 서버에 업로드되지 않음
- ✅ **빠른 처리** - 브라우저에서 직접 처리
- ✅ **다양한 형식 지원** - MP4, MOV, AVI, JPG, PNG 등

## 🚀 사용 방법

1. [블록툴툴 웹사이트](https://blogtooltool.com) 접속
2. 원하는 도구 선택 (GIF 변환기, 워터마크 추가, 블로그 썸네일)
3. 파일 업로드 및 설정 조정
4. 결과물 다운로드

## 🛠️ 기술 스택

- HTML5 Canvas API
- JavaScript (클라이언트 사이드 처리)
- Bootstrap 5 (반응형 디자인)
- GitHub Pages (호스팅)
